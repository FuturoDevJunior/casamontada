# CASA MONTADA - Landing Page

Esta é uma landing page responsiva para um serviço de montagem e desmontagem de móveis no Rio de Janeiro.

## Visão Geral

A landing page foi desenvolvida com foco em:

- Design moderno e responsivo
- Otimização para dispositivos móveis
- Velocidade de carregamento
- Experiência do usuário (UX)
- Conversão de visitantes em clientes

## Características

- Layout totalmente responsivo
- Animações suaves ao rolar a página
- Botão flutuante de WhatsApp para contato rápido
- Seções organizadas com informações claras sobre os serviços
- Depoimentos de clientes para aumentar a confiança
- Formulário de contato simplificado
- Compatibilidade cross-browser
- Performance otimizada

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e Flexbox/Grid)
- JavaScript (vanilla)
- Font Awesome para ícones
- Google Fonts para tipografia
- Animações CSS para melhor experiência do usuário

## Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em um navegador web
3. Para fazer alterações, edite os arquivos conforme necessário

## Personalização

### Cores

As cores principais do site são definidas como variáveis CSS no início do arquivo `styles.css`. Para mudar o esquema de cores, basta editar estas variáveis:

```css
:root {
    --primary-color: #ff6b00;
    --secondary-color: #2c3e50;
    --accent-color: #3498db;
    /* ... outras variáveis de cor ... */
}
```

### Informações de Contato

Para atualizar as informações de contato, edite os seguintes elementos no arquivo `index.html`:

1. Número de WhatsApp nos links com classe `.btn-whatsapp`
2. Número de telefone nos links com `href="tel:..."`
3. Endereço e área de atendimento nas seções de contato

## Responsividade

O design é totalmente responsivo e adaptável a diferentes tamanhos de tela:

- Desktop: 1200px e acima
- Tablet: 768px a 1199px
- Mobile: 767px e abaixo

## Performance

Para manter a performance otimizada:

- As imagens devem ser comprimidas antes de serem adicionadas
- O JavaScript é carregado no final do documento
- As fontes são pré-carregadas
- Animações são otimizadas para não afetar o desempenho

## 📷 Como Fazer Capturas de Tela do Site

Para adicionar imagens do site ao README:
1. Abra o arquivo `index.html` no navegador
2. Use a tecla `Print Screen` (Windows) ou `Command + Shift + 4` (Mac) para capturar telas
3. Salve as imagens na pasta `capturas` com os seguintes nomes:
   - `captura_inicial.jpg` - Página inicial/hero
   - `captura_servicos.jpg` - Seção de serviços
   - `captura_galeria.jpg` - Galeria de trabalhos
   - `captura_depoimentos.jpg` - Seção de depoimentos
   - `captura_whatsapp.jpg` - Botão flutuante de WhatsApp
